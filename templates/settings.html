Why These Settings Matter

Engineers are more likely to trust findings when they can tune the thresholds behind them.

Trust

A warning is easier to trust when the engineer knows exactly what threshold Atlas Intelligence used.

Adaptability

Different projects need different tolerances. Prototype boards and production boards should not always be judged the same way.

Active Config Impact

The thresholds below influence what findings are produced and what shows up in saved run detail pages.

Spacing Changes placement warnings
Power Nets Changes rail and return-path expectations
Trace Length Changes signal-path findings
Thermal Threshold Changes hotspot detection sensitivity
Ground Reference Changes EMI / return-path rule behavior
