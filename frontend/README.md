# Frontend

This is Silicore's primary UI application.

## Stack

- React
- TypeScript
- TanStack Start
- CSS/Tailwind-based styling

## Commands

```bash
npm install
npm run dev
npm run build
```

## Where To Build UI

Use this app for:

- new pages
- new components
- visual redesigns
- navigation and layout changes
- responsive behavior
- animation and interaction work

Avoid building new user-facing product screens in server-rendered Jinja templates unless there is a backend-only reason to do so.
